# Competitive Programming Solutions

A collection of competitive programming problem solutions organized by platform.

## Repository Structure

```
Competitive_Programmings/
├── Leetcode_problem_solution/     # All LeetCode problems
│   ├── [Problem Number]-[Name]/
│   │   ├── README.md             # Problem statement
│   │   └── solution.py           # Solution code
│   └── ...
│
├── Codeforce_problem_solution/    # All Codeforces problems
│   ├── basic/                    # Beginner level problems
│   │   ├── [Problem Name]/
│   │   │   ├── README.md
│   │   │   └── solution.txt
│   │   └── ...
│   └── ...
│
└── README.md                      # This file
```

## Languages Used

## Platforms
- **LeetCode**: Various difficulty levels

## How to Use

1. Navigate to the relevant platform folder
2. Find the problem by number or name
3. Each problem folder contains:
   - `README.md` - Problem description and examples
   - Solution files (`.py`, `.cpp`, `.txt`)

## Getting Started

To run any Python solution:
```bash
python3 solution.py
```

For C++ solutions:
```bash
g++ -o solution solution.cpp
./solution
```
